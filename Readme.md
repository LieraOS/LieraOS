## English

**What is LieraOS?**

LieraOS is an experimental Linux distribution designed for gamers and Windows users looking to switch. The goal is to provide a **fully configured, gaming-ready Linux system** that feels familiar to Windows users while leveraging all the advantages of Linux.  

**Key Features**

- **Pre-installed gaming launchers:** Steam, Heroic, GOG, and more  
- **Performance optimizations:** GameMode, VK Basalt, optimized AMD & Nvidia drivers  
- **Windows-like user interface:** Cinnamon desktop + integrated tools:
  - **LieraSnap** – flexible system and folder backups, bootable ISO options  
  - **LieraTask** – modern task manager  
  - **LieraDM (Device Manager)** – full hardware overview  
  - **LieraToolkit** – collection of useful tools for gamers and power users  
- **Live-CD + Installer:** Try without affecting your current system  
- **Safety net:** Backups and snapshots make updates and changes risk-free  

**Goal of LieraOS**

Many gamers hesitate to switch to Linux due to:  
- Fragmented launcher ecosystem  
- Complex driver installation  
- Lack of reliable backup solutions  
- Variety of Linux distributions  

LieraOS aims to remove these barriers and provide a **comfortable, immediately playable system**.  

**Contribute**

LieraOS is still in early development. If you want to get involved – coding, testing, UI design, or feedback – you are very welcome!  
- Report issues on GitHub  
- Pull requests are welcome  
- Ideas and suggestions are highly appreciated

## Deutsch

**Was ist LieraOS?**

LieraOS ist eine experimentelle Linux-Distribution, die speziell für Gamer und Windows-Umsteiger entwickelt wird. Ziel ist es, ein **fertig konfiguriertes, spielbereites Linux-System** bereitzustellen, das sich für Windows-User vertraut anfühlt, gleichzeitig aber die Vorteile von Linux voll ausnutzt.  

**Besondere Features**

- **Vorinstallierte Gaming-Launcher:** Steam, Heroic, GOG u.v.m.  
- **Performance-Optimierungen:** GameMode, VK Basalt, passende Treiber für AMD und Nvidia  
- **Windows-ähnliche Benutzeroberfläche:** Cinnamon-Desktop + eigene Tools:
  - **LieraSnap** – flexible System- und Verzeichnis-Backups, bootfähige ISO-Optionen  
  - **LieraTask** – moderner Task Manager  
  - **LieraDM (Device Manager)** – Gerätemanager für Hardwareübersicht  
  - **LieraToolkit** – Sammlung nützlicher Tools für Gamer und Power-User  
- **Live-CD + Installer:** Einfach ausprobieren, ohne das aktuelle System zu verändern  
- **Sicherheitsnetz:** Backups und Snapshots machen Updates und Änderungen risikofrei  

**Ziel von LieraOS**

Viele Gamer scheuen den Umstieg zu Linux wegen:  
- Fragmentierter Launcher-Landschaft  
- Komplexer Treiberinstallation  
- Fehlender Backup-Lösungen  
- Unterschiedlicher Distributionen  

LieraOS soll all diese Hürden beseitigen und ein **komfortables, sofort spielbares System** bieten.
